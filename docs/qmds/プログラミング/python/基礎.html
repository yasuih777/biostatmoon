インデックス・スライス

[]の中に整数値n(インデックス)を入れることで左からn番目の文字を取り出すことができる。

  • 数字の始まりは0であることに注意
  • 0: 3などの範囲(スライス)を指定することでその間の番号の文字列をまとめてとることができる
  • 右からのn番目の文字をとる場合は-nとする(こっちは数字の始まりは1)
print(text[0])
print(text[0: 5])
print(text[-1])

f-string

f""とすることで、括弧内に変数を入れることができる。文字列の生成でよく使うので覚えておくと良い。

for

for xxx in iteratorの後にインデントを揃えて記述したコードを反復することができる。

  • iteratorの部分に反復する対象のオブジェクトを指定する、反復処理はiterator内の要素全てに対して行われる

for i in range(5):

    print(i, c)

条件分岐

if-else

if xxxの後にインデントを揃えてコ―ドを記述することで、条件式xxxを満たす場合のみ処理を行うことができる。他に追加で条件分岐を行う場合にはelif yyy:とする、これら全ての条件を満たさない処理はelse:の後に記述する。

if a < 2:
    print(f"{a} is under than 2")
elif a < 0:
    print(f"{a} is negative value")
else:
    print(f"{a} is over than 2")

例外処理

try-except

try:の後の処理がエラーとなるかチェックされる。もしエラーがない場合はそのまま処理が実行されるが、エラーとなる場合はそこの処理以降のコードは実行されず、except:の後の処理が実行される。

try:
    print(f"check {a}")
    assert a is None
    print(f"{a} is None")
except:
    print(f"{a} is not None")
1

2

5 is not None

raise

raise xxxとすることで処理がエラーとなるパターンを自分で設定できる。xxxには組み込み例外関数を入力する。基本的にif-elseと同時に使い、dataclass等でパラメータの入力を制限するときに多用する。

if not isinstance(a, str):
    raise ValueError(f"{a} is not string")

def

よく使う処理はdefで関数化できる。関数には引数、戻り値を設定することができる。

クラス

クラスの基本文法

Pythonではクラスによって関数や(インスタンス)変数等をまとめてオブジェクト化することができる。次のようにしてクラスを定義することができる。

class HelloRobot:
    def __init__(self, name, age = 10):
        self.name = name
        self.age = age

    def greeting(self, lang = "english"):
        if lang == "english":
            print(f"Hello! My name is {self.name}, and age is {self.age}")
        elif lang == "japanese":
            print(f"こんにちは！私の名前は{self.name}です。年齢は{self.age}歳です。")

robot = HelloRobot("Robot")

print(robot.name)

robot.greeting()
robot.greeting("japanese")

2

3

4
クラス内での関数を定義できる
5

6

継承

定義済のクラスのオブジェクトを別クラスに継承することができる。Pytorchで深層学習を行う際には必須の技術である。

class SuperHelloRobot(HelloRobot):
    def __init__(self, name, your_name, age=10):
        super().__init__(name, age)

        self.your_name = your_name
    
    def farewell(self):
        print(f"ByeBye {self.your_name}!!")

robot = SuperHelloRobot("Super Robot", "Yugestu")

print(robot.age)
print(robot.your_name)

robot.greeting()
robot.farewell()
