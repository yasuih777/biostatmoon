Rstudio + renvを利用した環境構築

※このページのOSは以下とする。

OS: Windows11 WSL, Ubuntu 22.4

Rにおける環境構築

Rの環境構築はRStudioを、パッケージ管理はrenvを使うのがメジャーである。この記事でもこれらを使用した開発環境構築を行う。

  • RStudio: RのIDE、プロジェクト作成から対話モードによるプログラミングまで全てこれ一つで賄える: 公式リンク
    • WSL2環境なので、デスクトップ版ではなくRStudio Serverを使用する。
  • renv: パッケージ管理を行うことができるパッケージ: 公式リンク

RStudio Server, renvのインストール

あらかじめaptをアップデートしておく。

sudo apt update
sudo apt upgrade

RStudio Server

  1. Rのインストール
wget -qO- https://cloud.r-project.org/bin/linux/ubuntu/marutter_pubkey.asc | sudo tee -a /etc/apt/trusted.gpg.d/cran_ubuntu_key.asc
sudo add-apt-repository "deb https://cloud.r-project.org/bin/linux/ubuntu $(lsb_release -cs)-cran40/"
sudo apt install --no-install-recommends r-base

以下コマンドを実行。バージョン情報がでればOK

R --version
  1. RStudio Serverのインストール
wget https://rstudio.org/download/latest/stable/server/bionic/rstudio-server-latest-amd64.deb
sudo apt install gdebi
sudo gdebi rstudio-server-latest-amd64.deb

以下コマンドを実行。コマンド一覧がでればOK。

rstudio-server

renvのインストール

  1. Rを起動する
R
  1. Rの標準関数install.packagesでrenvをインストール
install.packages("renv")

ライブラリがインポートできるか確認

library(renv)
  1. Rを終了する
q()

Rstudio Serverの起動

  1. WSLでRstudio Serverを起動する
sudo rstudio-server start

※Rstudio Serverを終了する場合は以下のコマンドを実行する

sudo rstudio-server stop
  1. ブラウザからアクセス

こちらのリンクを開く

  1. ユーザー名とパスワードを入力

ブラウザの認証情報を入力するとRStudioの画面がでてきて、使えるようになる。

開発環境構築

プロジェクトの作成

  1. Rstudio Serverの右上のProject: (~)をクリックし、New Project...を選択する
  2. 「New Project Wizard」から、プロジェクトを作成するディレクトりを選択する
    1. 新しいディレクトリに作成する場合にはNew Directoryを選択する、その後画面でプロジェクト名と作成する階層を指定する
    2. 元々あるディレクトりに作成する場合にはExising Directoryを選択し、ディレクトリの場所を指定する
  3. Create Projectをクリック

作成後、<プロジェクト名>.Rprojというファイルが作成される。これにプロジェクトの情報が保存されていく。

renvの初期設定

Rコンソール上で行う。

  1. renvのインポートする
library(renv)
  1. renvの初期起動を行う
renv::init()

renvディレクトリとrenv.lockというファイルが作成される。これらのファイルにRのパッケージ情報が記録されていく。

renvによるパッケージ管理

上記の初期設定を行った後は、通常通りにパッケージのインストールや削除を行ってよい。その後、新しくインポートやバージョン更新してパッケージについて、以下のコマンドでrenvの情報を更新する必要がある。

renv::snapshot()

基本的には、作業がひと段落したときに上記のコマンドを実行するようにしておけば良い。

別のPCでパッケージ情報を再構築する際には以下のコマンドを実行する。

renv::restore()
