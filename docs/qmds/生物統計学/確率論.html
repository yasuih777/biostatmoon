確率論基礎

確率統計
Published

2024-01-24

Modified

2024-01-26

Abstract
統計学を扱う上での基礎となる確率論

事象と確率

事象

考察の材料とするための実験または観測を試行という。試行によって得られる個々の結果を標本点(または基本事象、単純事象、根元事元)という。標本点はいつも同じ値を取るのではなく、偶発的要因の影響を受けて揺らぎがある。この揺らぎを全て包含した標本点全体の集合を標本空間\(\Omega = \{ \omega_{1}, \omega_{2}, \cdots \}\)という。

Example

サイコロを振る試行における標本空間は次のようになる

\[ \Omega = \{ 1, 2, 3, 4, 5, 6 \} \]

今、試行の中で期待したい観測のみを抽出したいとする。これは標本空間の部分集合として表現でき、事象\(e \subset \Omega\)と呼ぶ。標本点\(\omega\)が事象\(e\)に含まれる(\(\omega \in e\))とき、事象\(e\)が生起したという。事象全体の集合を事象空間\(\mathcal{E}\)と呼ぶ。

  • 事象には基本的な集合演算(和集合、積集合、補集合)を適応することができる
  • 特殊な事象として、標本空間を事象とした全事象\(\Omega\)と、標本点を一つももたない空事象\(\varnothing\)がある
  • \(e_{1} \cap e_{2} = \varnothing\)のとき、事象\(e_{1}\)と事象\(e_{2}\)は互いに排反という
  • 事象空間の要素数は\(2^{|\Omega|}\)である
Example

サイコロから偶数が出る事象\(e_{odd}\)は次のようになる

\[ e_{odd} = \{ 2, 4, 6 \} \]

確率

標本点は偶発的要因により揺らぎをもって試行から生成されるとした。この試行を独立に十分な数\(n\)だけ行い、事象\(e\)が生起した回数を\(r\)とする。このとき Equation 1 を満たすような定数\(p\)の存在が(経験的に)確かであるとき、事象\(e\)を確率事象、\(p = P(e)\)を確率事象\(e\)の確率という。また、この写像\(P: \mathcal{E} \rightarrow [0, 1]\)を確率測度という。標本空間\(\Omega\)、事象空間\(\mathcal{E}\)、確率測度\(P\)の三つ組\((\Omega, \mathcal{E}, P)\)を確率空間と呼び、この空間をもって各種の確率理論を展開することができる。

\[ p = \lim_{n \rightarrow \infty} \frac{r}{n} \tag{1}\]

確率事象、確率はコルモゴロフの定理に従う。

コルモゴロフの定理
  1. \(e_{1}, e_{2}, e_{3}, \cdots\)が確率事象であるとき以下も全て確率事象である

\[ \begin{split} &e_{1} \cap e_{2} \cap e_{3} \cap \cdots \\ &e_{1} \cup e_{2} \cup e_{3} \cup \cdots \\ &e_{1}^{c}, e_{2}^{c}, e_{3}^{c}, \cdots \\ \end{split} \]

  1. 確率事象\(e_{1}, \Omega, \varnothing\)に対して以下が成立する

\[ \begin{split} &0 \leqq P(e_{1}) \leqq 1 \\ &P(\Omega) = 1 \\ &P(\varnothing) = 0 \\ \end{split} \]

  1. 確率事象\(e_{1}, e_{2}, e_{3}, \cdots\)は各々互いに素であるとき、以下が成立する

\[ P(e_{1} \cup e_{2} \cup e_{3} \cup \cdots) = P(e_{1}) + P(e_{2}) + P(e_{3}) + \cdots \]

条件付き確率

\(e_{1}, e_{2}\)が確率事象で、\(P(e_{2}) \neq 0\)のとき、 Equation 2 で定義される\(P(e_{1}|e_{2})\)を\(e_{2}\)の条件の元での事象\(e_{1}\)の(条件付き)確率という。

\[ P(e_{1} | e_{2}) = \frac{P(e_{1} \cap e_{2})}{P(e_{2})} \tag{2}\]

事象の独立

\(e_{1}, e_{2}\)が確率事象で、Equation 3 でが成り立つとき、\(e_{1}, e_{2}\)は互いに独立という。

\[ P(e_{1} \cap e_{2}) = P(e_{1}) P(e_{2}) \tag{3}\]

ベイズの定理

Equation 4 が\(i = 1, 2, \cdots, n\)に対して成り立つことをベイズの定理という。

\[ \begin{split} &e_{j} \cap e_{k} = \varnothing \ (j \neq k) \ かつ \ \Omega = e_{1} \cup e_{2} \cup \cdots \cup e_{n} \\ &\Rightarrow P(e_{i} | D) = \frac{P(e_{i})P(D | e_{i})}{\sum_{j=1}^{n} P(e_{j})P(D | e_{j})} \end{split} \tag{4}\]

確率変数・確率分布

確率変数

Equation 5 で定義される写像\(X\)を確率変数と呼ぶ(\(\mathcal{B}\)はボレル代数)。確率変数によって標本空間と事象空間を接合できる Figure 1 ため、ある試行から様々な事象や確率測度を展開することができる。また、この場合の確率測度\(P\)は\(P(X^{-1}(b))\)で定義され、「確率変数\(X\)は確率分布\(P\)に従う\((X \sim P)\)」という。主要な確率分布は別ページで扱う。

\[ X: \Omega \rightarrow \mathbb{R}, \exists X^{-1}(b) \in \mathcal{E} \ \mbox{for any} \ b \in \mathcal{B} \tag{5}\]

Example

サイコロ二個同時に振る試行について、次のような確率変数を考えることができる。

  • 出た目の総和
  • 偶数の目が出るサイコロの数
  • {サイコロAの目 / サイコロBの目}の値

また、確率変数は離散的なものと連続的なものの二つに分類することができる。次項で扱う。

確率分布

確率変数\(X\)がとりうる値が高々可算個、即ち\(x_{1}, x_{2}, \cdots, x_{n} \ (i \neq j \Rightarrow x_{i} \neq x_{j})\)の場合、\(X\)は離散的確率変数である。
また、Equation 6 で定義される\(f(x_{i})\)を確率分布という。

\[ P(X = x_{i}) = f(X_{i}) \tag{6}\]

確率分布の性質
  • \(f(X_{i}) \geqq 0 \ (i = 1, 2, \cdots)\)
  • \(\sum_{\mathbb{i}} f(x_{i}) = 1\)
  • \(P(X \in \mathcal{B}) = \sum_{x_{i} \in \mathcal{B}} f(x_{i})\)

確率密度関数

離散的でない確率変数\(X\)を連続的確率変数と呼び、Equation 7 で定義される\(f(x)\)を確率密度関数と呼ぶ。

\[ P(X \in \mathcal{B}) = \int_{\mathcal{B}} f(X) \tag{7}\]

この確率密度関数においては、確率素分\(f(x) dx\)に対して\(P(x \leqq X < x + dx) = f(x) dx\)が成り立つ。

確率密度関数の性質
  • \(P(X = x) = \int^{x}_{x} f(u) du = 0\)
  • \(f(x) \geqq 0\)
  • \(\int^{\infty}_{- \infty} f(u) du = 1\)
  • \(E \subset \mathcal{B}\)に対して、\(P(X \in E) = \int_{E} f(u) du\)

以降では連続的確率変数の定義を中心に扱うが、離散的確率変数の場合でもほぼ同等のことがいえる。

分布関数

確率変数\(X\)、実数$$xに対して Equation 8 を分布関数と呼ぶ。分布関数は様々な理論を展開するのによく用いられる。

\[ P(X \leqq x) = F(x) \tag{8}\]

分布関数の性質
  • 単調非減少関数
  • \(\forall a, b \in \mathbb{R}, P(a < X \leqq b) = F(b) - F(a)\)
  • \(F(- \infty) = 0, F(\infty) = 1\)
  • 右連続
  • \(F(x) = \int^{x}_{- \infty} f(u) fu\)

多次元の確率変数・分布

同時分布

複数の事象\(e_{1}, e_{2}\)が同時に生起する(\(e_{1} \cap e_{2}\))確率を同時確率という。これの確率変数\(X, Y\)に対応する関数\(f(x, y)\) Equation 9 を同時確率密度関数(離散の場合は同時確率分布)と呼ぶ。

\[ P((X, Y) \in \mathcal{E} \times \mathcal{E}) = \int_{X} \int_{Y} f(x, y) dy dx \tag{9}\]

特に、\([x_{1}, x_{2}] \times [y_{1}, y_{2}] \subset \mathcal{E} \times \mathcal{E}\)の場合、以下が成立する。

\[ P(x_{1} \leqq X \leqq x_{2}, y_{1} \leqq Y \leqq y_{2}) = \int^{x_{2}}_{x_{1}} \int^{y_{2}}_{y_{1}} f(x, y) dy dx \]

周辺化

確率変数\(X, Y\)に対して、同時確率密度関数\(f(x, y)\)のとりうる\(Y\)を総和することを「\(Y\)について\(f(x, y)\)を周辺化する」という Equation 10 。このとき周辺化された確率密度関数\(f(x)\)を周辺確率密度関数と呼ぶ。

\[ f(x) = \int_{Y} f(x, y) dy \tag{10}\]

条件付き分布

Equation 11 を「条件\(Y = y\)の元での\(X\)の条件付き確率密度関数」という。

\[ f(x | y) = \frac{f(x, y)}{f(y)} \tag{11}\]

確率変数の独立

独立な事象\(e_{1}, e_{2}\)に対する確率変数\(X_{1}, X_{2}\)が Equation 12 を満たすとき、\(X_{1}, X_{2}\)は独立であるという。

\[ f(x, y) = f(x) f(y) \tag{12}\]

Equation 11 より、独立な\(X_{1}, X_{2}\)に対しては以下が成立する。

\[ f(x | y) = f(x) \]

記述統計量

平均値

確率変数\(X\)の代表的、中心的な値としてよく用いられる平均値(期待値)\(E(X)\)は Equation 13 で定義される。

\[ E(X) = \int^{\infty}_{- \infty} x f(x) dx \tag{13}\]

平均値の性質
  • \(E(X_{1}, X_{2}, \cdots, X_{n}) = \sum^{n}_{i = 1} E(X_{i})\)
  • \(a, b\)が定数のとき、\(E(a X + b) = a E(X) + b\)
  • \(X, Y\)が独立のとき、\(E(X Y) = E(X) E(Y)\)
  • \(\phi(X) \sim P\)なる関数\(\phi\)について、\(E(\phi(X)) = \int^{\infty}_{- \infty} \phi(x) f(x) dx\)

また、\(Y = y\)なる条件の元でのXの条件付き平均値は Equation 14 で定義される。

\[ E(X | Y = y) = \frac{1}{f(y)} \int^{\infty}_{- \infty} x f(x, y) dx \tag{14}\]

分散

確率変数\(X\)のばらつきを表す値としてよく用いられる分散\(V(X)\)は Equation 15 で定義される。ただし、\(\mu = E(X)\)とした。

\[ V(X) = \int^{\infty}_{- \infty} (x -\mu)^{2} f(x) dx \tag{15}\]

分散の性質
  • \(a, b\)が定数のとき、\(V(a X + b) = a^{2} V(X)\)
  • \(V(X) = E(X^{2}) - E(X)^{2}\)

また、\(\sqrt{V(X)}\)を標準偏差と呼ぶ。

共分散・相関係数

二つの確率変数\(X, Y\)間の相関関係の強さを表す値としてよく用いられる(ピアソンの)相関係数\(R(X, Y)\)は Equation 16 で定義される。このときの\(C(X, Y)\)は共分散という。ただし、\(\mu_{x} = E(X), \mu_{y} = E(Y), \sigma_{x} = \sqrt{V(X)}, \sigma_{y} = \sqrt{V(Y)}\)とした。

\[ \begin{split} &C(X, Y) = E((X - \mu_{x})(Y - \mu_{y})) \\ &R(X, Y) = \frac{C(X, Y)}{\sigma_{x} \sigma_{y}} \end{split} \tag{16}\]

共分散・相関係数の性質
  • \(C(X, Y) = E(XY) - E(X) E(Y)\)
  • \(X, Y\)が独立のとき、\(C(X, Y) = R(X, Y) = 0\)
  • \(V(X_{1} + V_{2} + \cdots + X_{n}) = \sum^{n}_{i = 1} V(X_{i}) + 2 \sum_{i < j} C(X_{i}, X_{j})\)
    • 特に\(X_{1} + V_{2} + \cdots + X_{n}\)が独立のときは、\(V(X_{1} + V_{2} + \cdots + X_{n}) = \sum^{n}_{i = 1} V(X_{i})\)
  • \(|R(X, Y)| \leqq 1\)
    • 特に\(R(X, Y) = \pm 1 \Rightarrow \frac{X - E(X)}{\sqrt{V(X)}} \mp \frac{Y - E(Y)}{\sqrt{V(Y)}} = 0\)(複合同順)

相関係数には大まかに以下の意味がある。

  • 1に近いほど、「\(X\)が大きいならば\(Y\)も大きい」
  • -1に近いほど、「\(X\)が大きいならば\(Y\)は小さい」

これらの傾向をカットオフで分割し、 Table 1 のように考察することができる。ただし、このカットオフは一般的な例であることに注意する。
